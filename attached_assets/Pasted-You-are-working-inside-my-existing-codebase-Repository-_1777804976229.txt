- Show clear runtime errors if required env variables are missing during actual feature use.
- Do not log secret values.

If the app uses Next.js:
- Make sure `next build` succeeds.
- Avoid calling secret-dependent logic during static build unless guarded.
- Use dynamic rendering only where needed.
- Keep API secrets server-side only.

If the app uses Vite:
- Make sure `npm run build` succeeds.
- Use `import.meta.env` correctly.
- Only expose variables prefixed with `VITE_`.
